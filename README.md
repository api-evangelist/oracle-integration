# Oracle Integration (oracle-integration)
Oracle Integration provides native connectivity to Oracle and non-Oracle Software as a Service (SaaS) and On-premises applications, such as Oracle ERP Cloud, Oracle Service Cloud, HCM Cloud, Salesforce, Workday, EBS, SAP, NetSuite and others.

**URL:** [Visit APIs.json URL](https://www.oracle.com/integration/apis.json)
